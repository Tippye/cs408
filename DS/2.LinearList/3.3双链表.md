# 双链表

双链表就是在单链表的基础上，每个结点增加了前驱结点的地址，方便向前查找

插入、删除都与单链表有些区别（修改后继结点的时候把前驱结点也一块修改下


## 遍历
1. 后向遍历
    ```c++
    while (p != NULL) {
        // 某些处理
        p = p->next;
    }
    ```
2. 前向遍历
    ```c++
    while(p != NULL) {
        // 某些处理
        p = p->prior;
    }
    ```
3. 前向遍历（跳过头结点）
    ```c++
    while(p->prior != NULL) {
        // 某些处理
        p = p->prior
    }
    ```
