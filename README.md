# 程序调试

- 用Clion打开
- [CMakeLists](./DS/CMakeLists.txt)文件在`/DS`里面
- 运行可以在[/DS/main.cpp](./DS/main.cpp)中修改`#include`后面引入的文件
- 所有代码文件我都保存成头文件了，这样方便引入直接运行，不然每次都要注释CMakeLists太麻烦
- 文件格式用`.h`只是为了方便运行，可以改成`.cpp`文件单独运行，算法部分代码不变
